# SysPrior

Early public repository for the **SysPrior** project on system-prompt priority alignment in large language models.

## Current Status

This repository is being opened **before the full codebase and dataset release is finalized**.

At the current stage:

- the English paper draft is available;
- parts of the experimental pipeline and generated artifacts are present in the repository;
- the training code, evaluation scripts, and dataset packaging are still being cleaned up;
- file organization may continue to change as the project is prepared for a more complete public release.

Please treat the current repository as an **early research draft release**, not yet as a fully polished reproduction package.

## Paper Draft

The current English draft can be found here:

- PDF: `sysprior/paper_demo_en.pdf`
- LaTeX source: `sysprior/paper_demo_en.tex`

This draft is still under active revision. Experimental details, wording, figures, and references may be updated in later versions.

## What Is Included Right Now

- the current English manuscript draft;
- selected figures used in the draft;
- intermediate experiment outputs and analysis artifacts;
- partially organized project code used during benchmark construction and evaluation.

## What Is Not Fully Released Yet

The following components are **not yet in their final public form**:

- cleaned end-to-end training and evaluation pipeline;
- finalized benchmark packaging;
- fully documented dataset release;
- reproducibility scripts with stable public entry points;
- release-ready instructions for all experiments reported in the draft.

## Planned Updates

We plan to incrementally add:

1. a cleaner repository structure for public use;
2. documented evaluation scripts;
3. a better organized benchmark/data release;
4. more complete reproduction instructions;
5. updated paper drafts and supporting materials.

## Repository Note

Some files currently kept in the repository are research artifacts generated during drafting and experimentation. They are included for transparency, but they should not yet be interpreted as a finalized benchmark release.

## Citation

If you would like to reference this project before the formal paper release, please cite the repository link and mention that it corresponds to an early draft release. A formal citation entry will be added after the paper metadata is finalized.
