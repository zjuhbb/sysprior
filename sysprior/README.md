# Paper Draft Notes

This directory contains the current English paper draft and supporting manuscript assets for the **SysPrior** project.

## Main Files

- `paper_demo_en.pdf`: current compiled English draft
- `paper_demo_en.tex`: LaTeX source for the English draft
- `figure 1.png`, `metrics_comparison.png`, `cot_delta_subtype.png`: figures used in the draft

## Draft Notice

This manuscript is a **working draft**, not a camera-ready or archive-stable version.

In particular:

- the text may still be revised;
- references may continue to be expanded or reorganized;
- figures and experimental sections may be updated;
- some quantitative results may be replaced with cleaner reruns.

## Release Scope

The paper draft is being shared earlier than the full project release.
The associated code, benchmark packaging, and datasets are still under cleanup and may not yet match the final structure described in the manuscript.

If you are browsing this repository only for the paper, start with:

- `paper_demo_en.pdf` for reading
- `paper_demo_en.tex` for source inspection
